🧹 Clear Browser Cache

Klik tombol di bawah untuk membersihkan cache browser dan service worker.


📋 Manual Steps (Jika tombol tidak bekerja):

  1. Hard Reload: Tekan Ctrl + Shift + R (Windows) atau Cmd + Shift + R (Mac)
  2. Clear Cache via DevTools:
    • Tekan F12 untuk buka DevTools
    • Tab Application
    • Klik Clear storage di sidebar kiri
    • Centang semua opsi
    • Klik Clear site data
  3. Clear Browser Data: Tekan Ctrl + Shift + Delete → Clear cached images and files
  4. Incognito Mode: Buka halaman di Incognito/Private window
